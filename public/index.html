Diffology(alpha)

Diffology lets you synchronize data stored in Microsoft Access databases over the web without having to deal with complicated configurations, requiring technical or programming expertise, or significant changes to your current workflow.

Just Sync and you're done!

Features

  • Extremely simple to use! No setup!
  • Synchronize over the web — no need for network directory sharing.
  • Works with split and un-split databases.
  • Self-contained. Works with any firewall configuration.

Usage

  1. Exit Access and backup your database.
  2. Follow the link below and download and install Diffology. If you don't know which version of Office you're running, you can follow this guide.
  3. Next time you open up Access you'll have a new menu at the top.
  4. In the new menu, click Sync.
  5. Send your database to other collaborators. When you Sync, Diffology will upload your changes to the server and download changes introduced by other collaborators.

Download

  • Diffology v0.0.1 (for 64 bit Office)
  • Diffology v0.0.1 (for 32 bit Office)

Compatible with Microsoft Access 2007, 2010, 2013, 2016 and 2019.
