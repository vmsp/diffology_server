Diffology(alpha)

Diffology lets you synchronize data stored in Microsoft Access databases over the web without having to deal with complicated configurations, requiring technical or programming expertise, or significant changes to your current workflow. Just press Sync and you're done.

Features

  • Extremely simple to use! No setup!
  • Synchronized over the web — no need for network directory sharing.
  • Works with split and un-split databases.
  • Self-contained. Works with any firewall configuration.

Usage

Follow the download link below and install Diffology. Next time you open up Access, you will have a new menu at the top. When you want to synchronize your data just press Sync — the first sync will take extra time, subsequent syncs will be faster. Your database file is now ready to be shared! Just send it over to all collaborators. Diffology will upload your changes to the server and download changes introduced by other collaborators.
